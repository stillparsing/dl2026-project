4.3.5.3 MBRControl (M)
The MBRControl table is defined in [2], and Table 46 defines the Preconfiguration Data for the MBRControl table.
*MC1 means that only a limited set of DoneOnReset values is required to be supported by Opal SSC compliant
Storage Devices. Refer to section 4.3.5.3.1 for details.
Table 46 - Locking SP - MBRControl Table Preconfiguration
```json
[
    {
        "UID": "00 00 08 03 00 00 00 01",
        "Enable": "False",
        "Done": "False",
        "DoneOnReset": "Power Cycle *MC1"
    }
]
```
