5.6.4.11 Hash Method Group – HashInit (Object Method)
This method is used to initiate a hash “stream” using the hash object that invoked the method.  Only
one hash “stream” SHALL be able to be open at any one time for any individual hash object.
Invocation of this method is required before the Hash method MAY be successfully invoked.  In
preparation for beginning the hash “stream”, upon successful invocation of the Hashinit method, the
invoking hash object’s Accumulator column is set to zero.
