5.7.2.2.17 ContOnReset
This column value is a set of reset conditions. This value defines how a re-encryption process reacts to
reset conditions.
An empty set means the TPer_Reset_Stop condition is set for any reset condition. The
ReEncryptState value is set to PAUSED.
For each listed reset entry, the re-encryption process MAY continue after the associated reset is
detected.
