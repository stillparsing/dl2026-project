5.7.2.2.13 ReEncryptState
The value of this column identifies the currently applicable Re-encryption state (see 5.7.3.3).  The value
in the column affects the TPer’s response to the host’s requests in the ReencryptRequest column.
Reset configuration (ContOnReset) and a detected reset condition define the reported ReEncryptState
and PauseStatus values.
This column SHALL NOT be modifiable by the host.
