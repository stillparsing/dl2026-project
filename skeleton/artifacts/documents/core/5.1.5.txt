5.1.5 Method Status Codes
Begin Informative Content
SP method calls invoke specific operations and receive associated status.  The following sections
identify and define the status codes that are returned by the TPer in response to method invocations
and other operations.  Table 166 identifies the value associated with each of these status codes.
End Informative Content
Table 166 Status Codes
```markdown
| Name                 | Value  |
|----------------------|--------|
| SUCCESS              | 0x00   |
| NOT_AUTHORIZED       | 0x01   |
| OBSOLETE             | 0x02   |
| SP_BUSY              | 0x03   |
| SP_FAILED            | 0x04   |
| SP_DISABLED          | 0x05   |
| SP_FROZEN            | 0x06   |
| NO_SESSIONS_AVAILABLE| 0x07   |
| UNIQUENESS_CONFLICT  | 0x08   |
| INSUFFICIENT_SPACE   | 0x09   |
| INSUFFICIENT_ROWS    | 0x0A   |
| INVALID_PARAMETER    | 0x0C   |
| OBSOLETE             | 0x0D   |
| OBSOLETE             | 0x0E   |
| TPER_MALFUNCTION     | 0x0F   |
| TRANSACTION_FAILURE  | 0x10   |
| RESPONSE_OVERFLOW    | 0x11   |
| AUTHORITY_LOCKED_OUT | 0x12   |
| FAIL                 | 0x3F   |
```
